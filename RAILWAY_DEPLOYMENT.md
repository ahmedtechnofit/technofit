# 🚂 دليل نشر TechnoFit على Railway

## 📋 المتطلبات
- حساب GitHub
- المشروع مرفوع على GitHub ✅ (تم)

---

## 🔧 الخطوة 1: إنشاء حساب Railway

1. اذهب إلى: **https://railway.app**
2. اضغط على **"Start a New Project"** أو **"Login"**
3. اختر **"Login with GitHub"**
4. اسمح لـ Railway بالوصول إلى حسابك

---

## 🗄️ الخطوة 2: إنشاء قاعدة بيانات PostgreSQL

1. بعد تسجيل الدخول، اضغط **"New Project"**
2. اختر **"Provision PostgreSQL"**
3. ستظهر قاعدة البيانات في المشروع
4. اضغط على قاعدة البيانات لفتحها
5. اذهب إلى تبويب **"Variables"**
6. انسخ قيمة **DATABASE_URL** 

**شكل الرابط:**
```
postgresql://postgres:PASSWORD@HOST.railway.app:PORT/railway
```

---

## 🚀 الخطوة 3: نشر المشروع من GitHub

### الطريقة 1: من داخل Railway

1. في نفس المشروع، اضغط **"+"** (Add Service)
2. اختر **"GitHub Repo"**
3. اختر مستودع **"technofit"**
4. سيثبت Railway المشروع تلقائياً

### الطريقة 2: من GitHub (أسهل)

1. اذهب إلى: **https://railway.app/new**
2. اختر **"Deploy from GitHub repo"**
3. اختر مستودع **"ahmedtechnofit/technofit"**
4. اضغط **"Deploy Now"**

---

## ⚙️ الخطوة 4: إضافة متغيرات البيئة

1. اضغط على خدمة المشروع (اسم المستودع)
2. اذهب إلى تبويب **"Variables"**
3. اضغط **"New Variable"**
4. أضف المتغيرات التالية:

| الاسم | القيمة |
|-------|--------|
| `DATABASE_URL` | الرابط من قاعدة البيانات (من الخطوة 2) |

**💡 نصيحة:** يمكنك سحب المتغير من قاعدة البيانات:
- اضغط **"Add Reference"**
- اختر قاعدة البيانات
- اختر `DATABASE_URL`

---

## 🌐 الخطوة 5: الحصول على رابط الموقع

1. اضغط على خدمة المشروع
2. اذهب إلى تبويب **"Settings"**
3. ابحث عن **"Domains"**
4. اضغط **"Generate Domain"**
5. ستحصل على رابط مثل: `technofit.up.railway.app`

---

## ✅ الخطوة 6: التحقق من النشر

1. اذهب إلى الرابط الذي حصلت عليه
2. يجب أن يظهر موقع TechnoFit
3. جرب تسجيل الدخول كمشرف:
   - **Username:** admin
   - **Password:** TechnoFit@2024

---

## 🔍 استكشاف الأخطاء

### المشكلة: الموقع لا يعمل
**الحل:** 
1. اذهب إلى تبويب **"Deployments"**
2. تحقق من **"Build Logs"** و **"Deploy Logs"**
3. إذا كان هناك خطأ، انسخه وأرسله لي

### المشكلة: خطأ في قاعدة البيانات
**الحل:**
1. تأكد من أن `DATABASE_URL` صحيح
2. تأكد من أن قاعدة البيانات تعمل (لون أخضر)

### المشكلة: الموقع بطيء
**الحل:**
- Railway الخطة المجانية تضع التطبيقات في وضع sleep
- المرة الأولى ستكون بطيئة، ثم يصبح أسرع

## 📸 صور توضيحية

### 1. إنشاء مشروع جديد
```
New Project → Provision PostgreSQL
```

```
+ Add Service → GitHub Repo → اختر technofit
```

```
Variables → Add Reference → PostgreSQL → DATABASE_URL
```

### 4. توليد النطاق
